铁大秘书系统

  • 个人作品展示

铁大秘书系统，一个基于javabean+servlet+Mysql 的 MIS系统，这是我们参加创新创业大赛的一个项目，目前已获得软件著作权

铁大秘书系统

软件编写目的

随着商行业的迅速发展，网上交易的成交量也在飞跃式的增加，因此，快递行业迅速隆起，每天的快件量也在学校门前堆积如山。根据调查，我校正门口日均快递量基本在1000件左右，而每次快递员离开时，都会有很多快件依旧没有人认领，要么是因为忙没时间领取，要么是人不在校内无法领取，或者有的人干脆是躺在床上懒得去拿。我们就编写这个软件，为学生提供代取快递的服务，同时在我们的系统中，还有查看学生成绩，查看空闲教室的功能。

软件的开发与运行环境

开发环境：myeclipse10
开发支持：JDK1.7
操作系统：Windows Server 2012 R2
服务器：阿里云ECS
Web服务器：tomcat7.0
数据库：mysql5.1

运用的技术

javaBean servlet jquery
